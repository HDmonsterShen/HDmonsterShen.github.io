# 字符类

.匹配任意一个字符

t.
test->te

[]匹配括号中的任意一个字符

[t]e
test->te
[t]a
tad->ta
[te]s
tserses->ts&&es

-[]内表示字符串范围

[a-f]e
eesabae->ee&&ae

^位于[]的开头，匹配除括号中的字符串之外的任意一个字符串

[^a-f]e
eesabge->ge

# 数量限定符

?紧跟在它前面的单元应该匹配零次或一次

[0-9]?\.[0-9]
0.9->0.9
1.9->1.9
11.9->1.9
a.9->.9
[0-9]?\.[0-9]?
1.->1.

+紧跟在它前面的单元应该匹配一次或多次

[0-9]?\.[0-9]+
1.9->1.9
11.9->1.9
1.999->1.999

*紧跟在它前面的单元应该匹配零次或多次

[0-9]?\.[0-9]*
1.->1.
11.9->1.9
1.999->1.999

{N}紧跟在它前面的单元应该精确匹配N次

[0-9]{2}\.[0-9]*
1.
11.9->11.9
1.999

{N,}紧跟在它前面的单元格应该至少匹配N次

[0-9]{3,}\.[0-9]*1.11.9111.999->111.999

{N,M}紧跟在它前面的单元格应该至少匹配N次至多匹配M次

[0-9]{0,2}\.[0-9]*1.->1.11.9->11.9111.999->11.999

# 匹配小数

package main
import (	
    "fmt"	
    "regexp")
func main() {	
    str := "3.14 123.123 .68 haha 1.0 abc 7. ab.3 66.6 123."	
    ret := regexp.MustCompile(`[0-9]+\.[0-9]+`)    
    ret := regexp.MustCompile(`\d+\.\d+`)	
    alls := ret.FindAllStringSubmatch(str, -1)	
    fmt.Println(alls)}

# 匹配规则

# 普通字符匹配

如：/a/ 匹配字符 ‘a’，/a,b/ 匹配字符 ‘a,b’

# 转义字符匹配

\d 匹配一个数字，即0-9
\D 匹配一个非数字，即除了0-9
\w 匹配一个单词字符（字母、数字、下划线）
\W 匹配任何非单词字符。等价于[^A-Za-z0-9_]
\s 匹配一个空白符
\S 匹配一个非空白符
. 匹配一个任意字符

例：

var sTr01 = '123456asdf';
var re01 = /\d+/;
//匹配纯数字字符串
var re02 = /^\d+$/;
alert(re01.test(sTr01)); //弹出true
alert(re02.test(sTr01)); //弹出false

# 量词：对左边的匹配字符定义个数

? 出现零次或一次（最多出现一次）
+ 出现一次或多次（至少出现一次）
* 出现零次或多次（任意次）
{n} 出现n次
{n,m} 出现n到m次
{n,} 至少出现n次

# 任意一个或者范围

[abc123] : 匹配‘abc123’中的任意一个字符
[a-z0-9] : 匹配a到z或者0到9中的任意一个字符

# 限制开头结尾

^ 以紧挨的元素开头
$ 以紧挨的元素结尾

# 修饰参数

g： global，执行全局匹配（查找所有匹配而非在找到第一个匹配后停止）
i： ingore case，忽略大小写，默认大小写敏感

# 常用方法

test
用法：正则.test(字符串) 匹配成功，就返回真，否则就返回假
